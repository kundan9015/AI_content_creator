## 🔧 **TROUBLESHOOTING**

### **Problem 1: PC Offline Dikha**
```bash
# Solution:
1. Check agent running?
   Task Manager → python.exe

2. Manual start:
   python local-agent\agent.py

3. Check logs:
   notepad logs\app_*.log
```

### **Problem 2: Render App Sleep**
```
Normal hai! Free plan pe 15 min mein sleep hota hai.

Solution:
- Dashboard kholo → auto wakes up
- Or uptimerobot.com (free)
- Or paid plan ($7/mo) → never sleeps
```

### **Problem 3: Upload Slow**
```
Normal!
- Depends on internet speed
- Depends on file size
- 2GB video = 5-10 min

Solution:
- Use faster internet
- Or upload at night
- Or compress video first
```
